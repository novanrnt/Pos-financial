================================================================================
                        QUICK START
================================================================================

1. Generate Icons:
   - Use https://realfavicongenerator.net/
   - Or use ImageMagick/Node.js (see QUICK_COMMANDS.md)
   - Place in public/icons/

2. Build:
   npm run build

3. Test on iPhone:
   - npm run dev
   - Get your IP: ipconfig
   - On iPhone Safari: http://YOUR_IP:3000
   - Tap Share → Add to Home Screen
   - Launch and test

4. Test on Android:
   - npm run dev
   - On Android Chrome: http://YOUR_IP:3000
   - Menu → Install app
   - Launch and test
